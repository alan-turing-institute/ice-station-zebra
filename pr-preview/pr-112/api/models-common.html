TimeEmbed#

class ice_station_zebra.models.common.time_embed.TimeEmbed(dim=256, activation='SiLU')#

Bases: Module

Initialize the time embedding module.

This module takes pre-computed sinusoidal time embeddings (from _timestep_embedding()) and projects them through fully-connected layers to learn a more useful representation for the diffusion task. The MLP uses a dim→4*dim→dim expansion to provide additional flexibility for adapting the fixed sinusoidal embeddings.

Args:
dim (int, optional): Size of the input and output embedding dimension.

Defaults to 256.

activation (str, optional): Name of the activation function to use

(e.g., “SiLU”, “ReLU”). Defaults to “SiLU”.

Parameters:

  • dim (int)

  • activation (str)
