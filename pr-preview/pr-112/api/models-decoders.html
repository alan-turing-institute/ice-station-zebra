Decoders#

The decoders module provides classes for decoding latent space representations back to output data.

forward(x)#

Forward step: decode latent space into output space.

The default implementation simply calls self.rollout independently on each time slice. These are then stacked together to produce the final output.

Args:

x: TensorNTCHW with (batch_size, n_forecast_steps, n_latent_channels_total, latent_height, latent_width)
