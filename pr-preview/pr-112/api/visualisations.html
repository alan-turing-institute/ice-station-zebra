Visualisations#

The visualisations module provides plotting and visualization utilities for the Ice Station Zebra framework.

Functions#

Plotting Core#

ice_station_zebra.visualisations.plotting_core.levels_from_spec(spec)#

Generate contour levels from a plotting specification.

Creates evenly-spaced contour levels based on the minimum and maximum values specified in the PlotSpec. If vmin or vmax are not specified, defaults to 0.0 and 1.0.

Args:

spec: PlotSpec object containing vmin and vmax parameters.

Returns:

NumPy array of contour levels spanning from vmin to vmax with n_contour_levels steps.

Parameters:

spec (PlotSpec)

Return type:

ndarray

ice_station_zebra.visualisations.plotting_core.make_diff_colourmap(sample, *, mode)#

Construct colour mapping settings for a difference panel.

Behaviour depends on the difference mode:

  • “signed”: symmetric diverging scale centred on 0, useful for showing positive vs negative bias.

  • “absolute” / “smape”: sequential scale from 0 to max, useful for showing error magnitude.

Args:
sample: Either a full array of differences (for precompute mode)

or a scalar maximum difference (for two-pass mode).

mode: Difference mode (“signed”, “absolute”, or “smape”).

Returns:

DiffRenderParams: Normalisation, colour limits, and colourmap.

Parameters:

  • sample (ndarray | float)

  • mode (Literal['signed', 'absolute', 'smape'])

Return type:

DiffColourmapSpec

ice_station_zebra.visualisations.plotting_core.prepare_difference_stream(*, include_difference, diff_mode, strategy, ground_truth_stream, prediction_stream)#

General, reusable planner for animations (maps or time-series).

This function implements three different strategies for handling difference between ground truth and prediction in animations, each with different memory and computational trade-offs. The choice of strategy affects both memory usage and animation performance.

Args:

include_difference: Whether difference visualisation is requested. diff_mode: Type of difference computation (signed/absolute/smape). strategy: Strategy for difference computation:

  • “precompute”: Calculate all differences upfront

  • “two-pass”: Scan data to determine colour scale, then compute per-frame

  • “per-frame”: Compute differences on-demand

ground_truth_stream: 3D array of ground truth data over time. prediction_stream: 3D array of prediction data over time.

Returns:

Tuple of (difference_stream, colour_scale): - difference_stream: None unless strategy == ‘precompute’ - colour_scale: DiffColourmapSpec describing colourmap/norm/range

Parameters:

  • include_difference (bool)

  • diff_mode (Literal['signed', 'absolute', 'smape'])

  • strategy (Literal['precompute', 'two-pass', 'per-frame'])

  • ground_truth_stream (ndarray)

  • prediction_stream (ndarray)

Return type:

tuple[ndarray | None, DiffColourmapSpec | None]
